# Pomodoro Timer

## About
Keep track of your work and break time with custom intervals

## How To Run
### Online
Go to [rtflores.com/pomodoro](rotyflo.github.io/pomodoro)

### Locally
```
# Clone this repo
git clone https://github.com/rotyflo/pomodoro

# Go into the folder
cd pomodoro

# Install dependencies
npm install

# Run the app
npm start
```
